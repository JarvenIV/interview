2.1 方法使用

方法集合

这里将会是方法集合

1.接上文,我想知道今天再加上5天的日期呢?

xui.now(xui.now(), 5); //返回今天之后5天的日期;
xui.now(xui.now(), 5, true); //返回今天之后5天的日期(包含时分秒);
xui.now(xui.now(), -5); //是的,你也可以传负数,表示今天之前5天的日期.
xui.now(xui.now('2017/11/01'), -5); //嗯,你还可以自定义日期和天数,是不是非常棒!

2.我想验证手机号,咋办?很简单,只需要传入2个参数,验证类型(名字,手机号,邮件)和对应的str,返回一个boolean

xui.checkLegal(type, str);

3.想获取url的某个参数呢?

xui.queryString(str);

4.有时候后台传给我们一个数据库存放的日期,就需要前端来进行转换一下.第一个参数是传入的日期,第二个是Boolean,如果是true的话就精确到时分秒,否则就是年月日

xui.formatDate(date, bool);

5.有时候为安全着想,我们需要把字符串符合等编码一下,那么传入相应的字符串即可, codeStr为编码,decodeStr为解码.

xui.codeStr(str);
xui.decodeStr(str);

6.随机输出一个在xx到xx的范围内数字(包含边界)呢?

xui.randomNum(1, 10);  //参数是从小到大排的

7.记录用户行为轨迹get方式请求

  • 关于如何记录详情请戳如何记录用户行为点击.
/*
传入需要统计的参数列表(对象形式)和异步接口地址API.
不用担心跨域问题哦!
只能以get方式请求.
*/
xui.recordData({
    data: {
        name: 'xui.js',
        version: xui.version,
    },
    url: 'www.example.com?a=1&b=2',
});

8.有时候我们想操作一下cookie,但是原生的操作方法过于繁琐,所以便捷的方法显得格外重要.

说明: 更多cookie知识详见JS积累-cookie

提示: 只有火狐支持本地页面保存cookie，建议使用火狐测试。鼠标右键-查看页面信息-安全-查看cookie

//`cookie`名和value必传
xui.setCookie(name, value, days, path)

//传入需要的cookie名即可,没有返回空
xui.getCookie(name)

//传入需要的cookie名即可删除`cookie`
xui.removeCookie(name)
  • 更多方法详见xui.js,哈哈
